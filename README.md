A simple Quiz Application developed in Java that reads multiple-choice questions from a file and allows users to attempt them interactively. The program records answers, calculates the score, and displays the final result.

Features
Load quiz questions from a file

Present multiple-choice questions (A/B/C/D format)

Accept user input for answers

Track correct responses

Display final score at the end

Language Used
Java
